# Checklist · 知识图谱与认知地图融合视图

## 数据融合与渲染

- [x] 认知地图页面同时获取知识图谱数据（`GET /graph`）和认知地图数据（`GET /api/causal/graph`）
- [x] 知识图谱节点和认知地图节点正确合并去重，以 slug 为键
- [x] 知识图谱边 ID 以 `kg_` 前缀区分，因果边以 `ce_` 前缀，超边以 `he_` 前缀
- [x] 知识图谱边渲染为蓝色虚线样式（`#3b82f6`），线宽 1.5，透明度 0.6
- [x] 知识图谱边标签显示 `relation` 字段
- [x] 知识图谱边带有 `data(type) = 'knowledge'` 标记
- [x] 图例包含"知识图谱边"条目
- [x] 知识图谱节点和认知地图节点在同一画布上正确显示

## 边类型开关

- [x] 工具栏包含"知识图谱边"开关按钮
- [x] 工具栏包含"认知地图边"开关按钮
- [x] 两个开关默认均为开启状态
- [x] 关闭"知识图谱边"开关后，所有知识图谱边从画布上消失
- [x] 关闭"认知地图边"开关后，所有认知地图边从画布上消失
- [x] 两个开关可同时关闭（画布仅显示节点，无边）
- [x] 开关状态不影响节点的显示

## 融合聚类

- [x] 知识边以权重 0.5 参与连通分量计算

- [x] 建议输出标注模块类型（knowledge / causal / mixed）
- [x] 混合模块建议以特殊样式显示
- [x] 建议卡片包含"类型"标签

- [x] 导出 PNG 时正确反映当前边类型开关状态
