# 理想 AI 知识库

欢迎使用 **Alethia 认知共生知识库**。这是一个由 AI 驱动的知识管理系统，能够自动从文档中提取知识、构建实体图谱，并通过对话问答的方式帮助你探索和理解知识。

## 快速开始

- 🔍 **搜索知识**：使用顶部搜索栏快速查找相关概念和文档
- 💬 **智能问答**：点击右上角「快速提问」与 AI 对话，获取带引用的答案
- 📊 **知识图谱**：浏览「图谱」页面，直观查看实体之间的关联
- ✅ **审核变更**：在「审核」页面查看 AI 提取的知识变更并确认

## 核心功能

| 功能 | 说明 |
|------|------|
| 多模态摄入 | 支持 PDF、Word、Excel、网页、音视频等多种格式 |
| 自动知识提取 | AI 自动从文档中提取实体、关系和事实 |
| 混合检索 | 向量 + 全文 + 图谱的多维度检索 |
| 溯源问答 | 每个答案都附带可追溯的来源证据 |
| 自进化 | 系统会根据使用反馈持续优化知识库质量 |
